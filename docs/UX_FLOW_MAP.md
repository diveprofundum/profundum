# UX Flow Map (Draft)

## 1) Onboarding → BLE permissions
1. Launch app
2. Intro screens: value prop + privacy
3. Request BLE permissions
4. Show nearby devices list
5. Select device → pair

## 2) First sync → first dive
1. Sync progress screen
2. Show results summary (dives imported, last dive date)
3. Go to Dive List (filtered to recent imports)
4. Tap a dive → Dive Detail

## 3) Dive detail → segment workflow
1. Open depth profile
2. Drag to select a range
3. Segment action bar: name + tags
4. Save segment
5. Segment stats panel updates

## 4) Formula creation → list columns
1. Open Formula Library
2. Create new formula
3. Validate and preview
4. Apply to all dives (or filtered)
5. Add as column in list view

## 5) Settings → time format
1. Open Settings
2. Display section
3. Toggle time format
4. Preview updates in list and detail views

## 6) Failure and recovery (BLE)
1. Sync fails → show error state
2. Offer retry and diagnostic tips
3. Resume from last downloaded dive

## 7) Data management
1. Edit site/buddy/equipment
2. Associate with dive
3. Use in filters
